# Sree Ahillan — Personal Portfolio

A personal portfolio website built with **pure HTML5 and CSS3** — no frameworks, no JavaScript.

## Features
- Responsive design with multiple breakpoints (mobile, tablet, desktop)
- Smooth scroll navigation
- Luxury dark theme with purple accent palette
- Sections: Hero, About, Projects, Resume, Certifications, Contact

## Sections
- **Hero** — Name, role, availability badge, and stats (CGPA, Projects, Technologies)
- **About** — Bio, skills list, and interests
- **Projects** — 4 projects with tech stack tags and GitHub links
- **Resume** — Experience and education timeline
- **Certifications** — HackerRank, Udemy bootcamp, English course
- **Contact** — Email, LinkedIn, GitHub, and phone

## Tech Stack
- HTML5
- CSS3 (Flexbox, Grid, Media Queries)
- Google Fonts (Syne + Space Grotesk)
